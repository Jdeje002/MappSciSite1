Create DS Output Source

The need to find the data structure members' values that go in and out of business functions often requires analysis of the Oracle JDE log files.

Analysts tasked with the job know how difficult it is to decipher the entries in the Oracle JDE log files. To speed up the process of finding the root cause, analysts can use business functions that output the data members' values to easier-to-read custom log files.

Easier-to-read files are time savers for proof-of-concept, testing, and trouble shooting. MappSci can reduce up to 95% of the mostly repetitive effort that would be required to create these business functions.

The process starts with the customer's sending a header file that contains the data structure and culminates with MappSci's sending a text file that contains source code and "copy and paste "instructions. A JDE technical person can then use the file to complete the task of compiling the business function.

On the average, the business function can be of use 35 minutes after MappSci receives the header file from the customer.
